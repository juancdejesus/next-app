# Users Folder

This folder contains all the components related to the users page.

- `page.tsx`: The main component for the users page.
- `UserAvatar.tsx`: A component to display the user's avatar.
- `UserForm.tsx`: A component to create and edit users.
- `UserTable.tsx`: A component to display the list of users.
- `UserToolbar.tsx`: A component with the actions for the users page (search, add user).
